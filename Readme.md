### Access Apollo Server
``` bash
http://localhost:8000/graphql
```

### List down all the docker containers
``` bash
docker ps
```

### Enter into your container using bash
``` bash
docker exec -it [container-name] bash
```

### Switch user
``` bash
su [username]
```

### For postgres cmd
``` bash
psql
```

### List all the databases
``` bash
\l
```

### Use any database
``` bash
\c [db name]
```

### describle table
``` bash
\dt
```

### See data
``` bash
SELECT * FROM [table name];
```
